# Juego-Musica
Este proyecto consiste en un juego en el cual se le presentará al usuario una escala musical incompleta. El usuario deberá seleccionar las opciones correctas para completar la escala.
Una vez seleccionadas todas las respuestas podrá revelar la respuesta y comparar sus respuestas con la solución final.
El usuario deberá poder crearse una cuenta para poder guardar su progreso.
Deberá poder ver estadísticas como:
    . Ejercicios completados
    . Respuestas correctas
    . Respuestas incorrectas

El usuario deberá poder modificar su nombre de usuario y eliminar su cuenta cuando lo desee.
